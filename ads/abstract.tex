\begin{otherlanguage}{english}
\begin{abstract}
Othello is a board game for two players. Due to the games deterministic nature, the strength of a player is solely
determined by their decision making and strategy.

The goal of this research project is to develop an artificial intelligence for the game Othello, which aims to be as
strong as possible, ideally beating human players as well as some publicly available artificially intelligent agents. This
artificial intelligence is to be implemented as a Jupyter Notebook in the Python programming language, and can be played
against by a human.

The MiniMax-algorithm with depth-limited search and a heuristic for evaluation Game States serves as a base for the
implementation, which is improved upon in terms of performance and therefore strength by using optimisations such as
AlphaBeta-pruning and probabilistic forward cuts using ProbCut. In addition, different features for use in the heuristic
function are tested and weighted.

\end{abstract}
\end{otherlanguage}

\begin{abstract}
Othello ist ein Brettspiel, welches von zwei Spielern gespielt wird. Da das Spiel völlig deterministisch ist, wird die
Stärke eines Spielers einzig und allein durch dessen Entscheidungsfällung und Strategie bestimmt.

Ziel dieser Arbeit ist es, eine möglichst starke
Künstliche Intelligenz für das Spiel Othello in Form eines Jupyter Notebooks in der Programmiersprache Python zu
entwickeln, gegen die ein menschlicher Spieler spielen kann.

Grundlage für die Umsetzung stellt der MiniMax-Algorithmus mit begrenzter Tiefensuche unter anwendung einer
heuristischen Evaluationsfunktion dar. Dessen Performance und somit Stärke wird durch die Verwendung von
AlphaBeta-Pruning und einer statistischen Abschätzung durch ProbCut weiter verbessert. Außerdem beschäftigt sich die
Arbeit mit der Findung und Gewichtung geeigneter Merkmale zur Berechnung der Heuristik.
